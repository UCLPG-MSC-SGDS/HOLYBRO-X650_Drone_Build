# HOLYBRO X650 Quadcopter Drone Project

Welcome, this project is aimed at building a powerful, fully automated quadcopter drone from scratch. The goal is to develop a platform capable of performing basic remote sensing and automated field surveying task through photogrammetry, and thermal imaging.

A significant part of this project involves a lot of experimentation and getting a feel for the various components (Raspberry Pi 5, SIYI remote controller, Pixhawk 6C Flight Controller Unit, Collision sensors and so on) to evaluate performance and ensure seamless integration within the drone’s frame (HOLYBRO X650). Expect extensive testing, iteration, and trial and error throughout the development process. All stages of the build will be thoroughly documented through written notes and video footage, with the aim of making the entire process open and accessible to anyone interested in drone technology.

The process for creating this 650mm diameter quadcopter is experimental serving as a learning experience as the long-term aim beyond this project will be to build another drone with top-tier and high-end components for performing much complex tasks.
